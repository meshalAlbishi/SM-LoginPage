# SM-LoginPage

Simple Login page using html, css, js, jQuery, ajax, php, and mySQL.

## Database
Only one simple table called 'users', with the following column

| column name  | type |
| ------------- | ------------- |
| id | number |
| email | varchar(100) |
| password | varchar(100) |
